<p>Each BC produces a compiled binary.</p>  <p>Operations defines a number of cross-cutting behaviors, e.g. logging, authentication, authorization, etc.</p>  <p>A set of operations build/deployment scripts bundles the compiled BC binary along with the binaries from the respective Operations ACs.&# 160; Then, a new config and EndpointConfiguration is defined.&# 160; Finally a deployment package is produced.</p>
